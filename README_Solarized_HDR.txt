Ukiyo-e Inspired Desktop Backgrounds (Solarized Theme)

This package contains traditional Japanese ukiyo-e style desktop backgrounds adapted to modern themes:
- Plant Zoo
- Dinosaurs
- Harry Potter
- Outer Space

Versions included:
- Standard and HDR formats
- 2560x1440 (16:9 laptop resolution)
- 3840x2160 (4K resolution)

Scripts:
- set_wallpaper_mx.sh: Basic HDR-aware rotation for X11 (1440p)
- set_wallpaper_mx_4k.sh: HDR + resolution-aware rotation for X11 (4K and 1440p)

Place all files in ~/Pictures/Wallpapers and run the script of your choice.
